## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/Smart-Personal-Finance-Tracker.git
```

---

## Move into Project

```bash
cd Smart-Personal-Finance-Tracker
```

---

## Create Virtual Environment

### macOS/Linux

```bash
python3 -m venv venv
```

Activate

```bash
source venv/bin/activate
```

### Windows

```bash
python -m venv venv

venv\Scripts\activate
```

```bash

```

```bash

```

```bash
streamlit run app.py
```
